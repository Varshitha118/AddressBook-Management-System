The Address Book Management System is a simple yet efficient application developed in C programming language to store, manage, and organize contact details. This project demonstrates strong understanding of data structures, file handling, and modular programming.

🚀 Features
    ➕ Create new contacts (Name, Phone Number, Email.)
    🔍 Search contacts
    ✏️ Edit existing contact details
    ❌ Delete contacts
    📋 Display all saved contacts
    💾 Persistent storage using file handling
    
🛠️ Technologies Used
   * C Programming
   * Data Structures
   * File Handling
   * Modular Design

💡 Project Highlights
    * Efficient handling of records using structured data
    * User-friendly menu-driven interface
    * Optimized searching and updating operations
    * Clean and maintainable code structure
    
🎯 Use Case
This project can be used as a basic contact manager for personal use and serves as a strong foundation for building more advanced applications like mobile contact apps or database-driven systems.
